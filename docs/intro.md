# Introduction

Wirestate is a state management framework based on InversifyJS dependency injection.
It organizes application logic into injectable services that communicate through events, commands, and queries.

## Key Concepts

- **Services**: Injectable classes containing business logic.
- **Events**: One-to-many communication between services.
- **Commands**: Request to perform an action.
- **Queries**: Request to retrieve data.
